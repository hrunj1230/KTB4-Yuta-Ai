# LuckyDate

전통 역법 기반 이사 날짜 추천 CLI 프로그램

## 소개

LuckyDate는 한국 전통 역법과 택일 개념을 기반으로
이사나 개업 등에 적합한 날짜를 추천해주는 CLI 프로그램입니다.

현재 구현 기능:
- 손없는날 계산
- 음력 변환
- 띠 충돌 검사
- 날짜 추천 시스템

## 실행 방법

### 설치

```bash
pip install korean-lunar-calendar
```

---

### 실행

```bash
python3 week01.py move 19951230
```

---

## 예시 출력

```txt
생년월일: 19951230
2026-08-31가 이사에 좋은 날짜 입니다.
2026-10-30가 이사에 좋은 날짜 입니다.
2026-12-17가 이사에 좋은 날짜 입니다.
2027-02-15가 이사에 좋은 날짜 입니다.
2027-04-16가 이사에 좋은 날짜 입니다.
```

## 사용 패키지

- argparse
- datetime
- korean-lunar-calendar

---

## 내부 로직

1. 사용자 생년월일 입력
2. 사용자 지지 계산
3. 오늘부터 1년 날짜 탐색
4. 음력 변환
5. 손없는날 검사
6. 날짜 지지 계산
7. 띠 충돌 제거
8. 추천 결과 출력
